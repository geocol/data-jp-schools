NAME

Geo::Area::Jp::Prefectures - List of Japanese prefectures

SYNOPSIS

  require Geo::Area::Jp::Prefectures;
  warn @Geo::Area::Jp::Prefectures::JpanLongName;
  warn @Geo::Area::Jp::Prefectures::JpanShortName;
  warn @Geo::Area::Jp::Prefectures::LatnLongName;
  warn @Geo::Area::Jp::Prefectures::LatnShortName;

DESCRIPTION

The Geo::Area::Jp::Prefectures module provides lists of names of prefectures in Japan in various forms.
